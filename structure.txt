1. pyinstaller로 stock-on-monitor.exe 빌드
2. pyinstaller --add-data로 stock-on-monitor 내장해서 stock-on-monitor-installer.exe 빌드
3. 사용자는 stock-on-monitor-installer.exe 하나만 받으면 됨
4. installer.exe 실행
   → 설치 경로 지정 (예: C:\Program Files\StockOnMonitor\)
   → main.exe를 해당 경로에 복사
   → config.json, stocks.json 생성
   → 바로가기 생성
   → 설치 경로의 main.exe 자동 실행
   → installer.exe 종료

# 1. dist 폴더에 stock-on-monitor.exe 준비
pyinstaller --onefile --noconsole -n stock-on-monitor main.py

# 2. PyInstaller로 installer.exe 빌드
pyinstaller --onefile --add-data "dist/stock-on-monitor.exe:." --noconsole -n stock-on-monitor-installer installer.py
